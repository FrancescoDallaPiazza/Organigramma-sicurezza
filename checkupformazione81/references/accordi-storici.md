# Accordi storici — indice di stratificazione normativa

Da consultare **per prima cosa** quando si analizza un attestato: serve a capire quale norma
si applicava alla data di emissione e quale si applica oggi.

## Quale norma si applica a un attestato emesso in data X

| Periodo di emissione | Norma applicabile (formazione) | File di riferimento |
|---|---|---|
| fino al 23/05/2025 | Accordi 2011/2012/2016 (lavoratori, DL-RSPP, attrezzature, RSPP/ASPP) | `accordi/ASR_221_2011_lavoratori_RAW.txt`, `accordi/ASR_223_2011_DL_RSPP_RAW.txt`, `accordi/ASR_53_2012_attrezzature_RAW.txt`, `accordi/ASR_128_2016_RSPP_ASPP_RAW.pdf` |
| periodo transitorio 24/05/2025 – 23/05/2026 | corsi ammessi sia nel vecchio formato sia nel nuovo ASR | entrambi |
| dal 24/05/2025 (a regime) | **ASR 17/04/2025** (Rep. Atti 59/CSR) | `accordi/ASR_59_2025_nuovo_accordo_RAW.txt` |

## Norme per materia

- **Lavoratori (vecchio regime)** → `accordi/ASR_221_2011_lavoratori_RAW.txt`
- **Datore di lavoro-RSPP (vecchio regime, 16/32/48h)** → `accordi/ASR_223_2011_DL_RSPP_RAW.txt`
- **Attrezzature art. 73 c. 5 (vecchio regime)** → `accordi/ASR_53_2012_attrezzature_RAW.txt`
- **Linee applicative 2012** → `accordi/ASR_153_2012_linee_applicative_RAW.pdf` (scansione)
- **RSPP/ASPP (vecchio regime, confermato dal 2025)** → `accordi/ASR_128_2016_RSPP_ASPP_RAW.pdf` (scansione)
- **Nuovo accordo unico 2025** → `accordi/ASR_59_2025_nuovo_accordo_RAW.txt`
- **Antincendio vigente** → `DM_02_09_2021_ANTINCENDIO.txt` (abrogato il DM 10/03/1998 → `DM_10_03_1998_ANTINCENDIO_ABROGATO.txt`)

- **Requisiti formatori** → `DI_06_03_2013_REQUISITI_FORMATORI.txt`
- **HACCP/alimentaristi** → `REG_CE_852_2004_HACCP.txt` + normativa regionale

## Date chiave ASR 17/04/2025
- **24/05/2025**: entrata in vigore.
- **24/05/2026**: fine periodo transitorio (corsi in vecchio formato).
- **24/05/2027**: termine per la prima formazione obbligatoria dei datori di lavoro già in carica.
